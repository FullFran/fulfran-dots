# My dotfiles (built on fulfran-dots)

Generated by [fulfran-dots](https://github.com/FullFran/fulfran-dots).

## Safer adoption pattern

If you already have dotfiles you want to keep, start with the `minimal` preset and opt-in per tool:

```nix
modules = [
  fulfran-dots.homeManagerModules.minimal  # core + shell only
  ./hosts/my-host.nix
];
```

Then disable any module whose config you want to preserve:

```nix
# hosts/my-host.nix
programs.fulfran.tmux.enableConfig = false;      # keep your tmux.conf
programs.fulfran.ghostty.enableConfig = false;   # keep your ghostty config
programs.fulfran.nvim.enableConfig = false;      # keep your nvim
programs.fulfran.shell.enableBash = false;       # keep your .bashrc
programs.fulfran.shell.enableZsh = false;        # keep your .zshrc
programs.fulfran.lazygit.enableConfig = false;   # keep your lazygit
```

Once you are comfortable with a module, remove the toggle (defaults to `true`) and let fulfran-dots manage it.
Switch to `full` when you are happy with all modules.

## Activate

```bash
nix run github:nix-community/home-manager -- switch --flake .#user@hostname -b backup
```

`-b backup` saves any overwritten files as `.backup` so you can recover them.
