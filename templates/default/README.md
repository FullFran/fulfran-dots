# My dotfiles (built on fulfran-dots)

Generated by [fulfran-dots](https://github.com/FullFran/fulfran-dots).

To disable a module feature, set the toggle in your host file:

```nix
programs.fulfran.tmux.enableConfig = false;  # bring your own tmux config
programs.fulfran.nvim.enableConfig = false;  # bring your own neovim config
programs.fulfran.dev.enableJdk = true;       # opt-in to jdk21
```

Run `home-manager switch --flake .#user@hostname -b backup` to activate.
